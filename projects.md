# Advent Calendar

This is the best advent calendar i have ever made as of writing this! It is also the only one i have made. 

## Features

- **URL-based Configuration**: The calendar works by taking a long string of characters from the URL and doing magic with it or something idk
- **Edit Mode**: There is an editmode for editing and a normal mode for normal things
- **HTML Support**: You can format the text in the doors by using HTML
- **Link Support**: The calendar currently doesn't support images in the door except if they are links

## How to Use

Get a string from someone else and put it at the end of the url or make your own using edit mode.
